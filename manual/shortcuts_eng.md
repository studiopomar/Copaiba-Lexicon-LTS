# Defaults and Commands Manual for Copaiba Lexikon

This document lists all keyboard shortcuts and commands available in the Copaiba Lexikon software.

## üìÇ File and Projects

| Action | Shortcut | Description |
| :--- | :--- | :--- |
| **Open voicebank...** | `Ctrl + O` | Opens a voicebank folder and loads `oto.ini` |
| **Open project...** | `Ctrl + Shift + O` | Opens a saved project file |
| **Save project** | `Ctrl + Shift + P` | Saves current state as a project |
| **Save (oto.ini)** | `Ctrl + S` | Saves changes to the `oto.ini` file |
| **Save as...** | `Ctrl + Shift + S` | Saves `oto.ini` with another name |
| **Reload** | `Ctrl + F5` | Reloads settings and current file |
| **Open voicebank folder** | `Ctrl + P` | Opens file explorer in the voicebank folder |
| **General Settings** | `Ctrl + ,` | Opens the settings window |

## ‚úèÔ∏è Editing

| Action | Shortcut | Description |
| :--- | :--- | :--- |
| **Undo** | `Ctrl + Z` | Undoes the last change |
| **Redo** | `Ctrl + Y` | Redoes the last undone change |
| **Rename Alias** | `Ctrl + R` | Renames the selected alias |
| **Duplicate Alias** | `Ctrl + I` | Duplicates the selected alias |
| **Delete Alias** | `Ctrl + D` | Removes the selected alias |
| **Mark as Done** | `Ctrl + M` | Toggles alias completion status |
| **Copy** | `Ctrl + C` | Copies selected cells in the table |
| **Paste** | `Ctrl + V` | Pastes data into the table |

## üåä Waveform and Navigation

### Keyboard (Waveform Focus)

| Key | Function | Description |
| :--- | :--- | :--- |
| **Q** | Set Offset | Sets Offset point at mouse position |
| **W** | Set Overlap | Sets Overlap point at mouse position |
| **E** | Set Preutterance | Sets Preutterance point at mouse position |
| **R** | Set Consonant | Sets Consonant point at mouse position |
| **T** | Set Cutoff | Sets Cutoff point at mouse position |
| **Up Arrow** | Previous Alias | Navigates to previous alias in list |
| **Down Arrow** | Next Alias | Navigates to next alias in list |
| **Mouse Scroll** | Navigate Aliases | Scroll up/down changes current alias |

### Mouse and Zoom

| Action | Shortcut | Description |
| :--- | :--- | :--- |
| **Horizontal Zoom** | `Ctrl + Scroll` | Increases or decreases time zoom |
| **Vertical Zoom** | `Alt + Scroll` | Increases or decreases wave amplitude |
| **Horizontal Pan** | `Shift + Scroll` | Moves waveform left or right |
| **Play Sector** | `Left Click` | Plays clicked sector (if "Play sector on click" is active) or main segment |

| Action | Shortcut | Description |
| :--- | :--- | :--- |
| **Play Segment** | `Space` | Plays configured region of current alias |
